# Personal Meetly

A privacy-first, self-hosted AI meeting assistant built as an MIT-licensed derivative of [Meetily Community Edition](https://github.com/Zackriya-Solutions/meetily).

## What this is

A personal fork that adds Pro-equivalent features (custom templates, PDF/DOCX export, auto-detect meetings, speaker diarization, calendar integration, Notion/Obsidian sync, meeting chat) as **original code** on top of the open-source MIT base. No license server, no telemetry, no vendor lock-in.
